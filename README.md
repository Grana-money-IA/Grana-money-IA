# IA Premium - Chat Inteligente

IA Premium é um chatbot inteligente com pagamentos únicos e assinatura via Stripe.

## Funcionalidades
- Chat inteligente
- Geração de conteúdo automatizada
- Pagamentos via Stripe (único ou assinatura)
- Interface simples e responsiva

## Estrutura do Projeto
/api
  └─ create-checkout-session.js
.env
index.html
premium.html
istyle.css
script.js
escript.js
README.md

## Instalação
1. Clone o repositório
2. Configure variáveis de ambiente (STRIPE_SECRET_KEY, STRIPE_PUBLIC_KEY, PRICE_ID_UNICO, PRICE_ID_ASSINATURA)
3. Faça deploy (Vercel recomendado)
